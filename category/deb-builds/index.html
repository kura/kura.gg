apt.kura.gg

After releasing my own versions of haproxy with SPDY support and nginx with ngx_pagespeed and SPDY support, I decided it would make sense to actually host these in my own apt repository, so IÂ did.

You can enable this by adding it to your aptÂ config.

wget -qO - https://apt.kura.gg/apt.kura.gg.key | sudo apt-key add -
echo "deb https://apt.kura.gg/ `lsb_release -cs` main" | sudo tee /etc/apt/sources.list.d/apt.kura.gg.list
sudo apt-get update

Simple.
