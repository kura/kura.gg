Hauntr - A theme for Pelican

Contents

Contents

  • Introduction
  • Features

(Image by Kawiku)

Introduction

Hauntr is a minimal, lightweight and clean theme for the Pelican blogging platform.

It is named after the Pokémon ‘Haunter’ because it is a modified version (you might say evolved) of my previous theme, Ghastly.

Features

  • Disqus
  • Pygments
  • CSS minifying using webassets
  • Share buttons
  • Custom 404 page
